Séries http://series.hol.es

Site de séries que lista os episódios das respectivas temporadas.

Linguagens

Na construção do site foram usadas as seguintes linguagens:

HTML

A estrutura da página.

CSS

Usado para estilizar os elementos da página.

JavaScript & jQuery

Ambos usados para manipulação dos elementos da página, tais como, transição de telas, comportamento dos tiles, etc.

Ajax

Usado para obter os dados das séries.

PHP

Usado como intermediário entre as requisições Ajax e o banco de dados.

MySQL

Usado como banco de dados das informações das séries.

Google Fonts

Usado para importar a fonte Lato.

Ferramentas

No desenvolvimento foram utilizadas as seguintes ferramentas e softwares:

Sublime Text

criação e edição de código.

FileZilla

envio dos arquivos para o servidor através de FTP.

Adobe Photoshop

edição de imagens.

Google Chrome, Mozilla Firefox e Internet Explorer

testes de layout, navegação, etc.
